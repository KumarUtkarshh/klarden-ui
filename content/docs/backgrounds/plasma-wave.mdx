title: Plasma Wave

category: Backgrounds
---

The Plasma Wave component creates mesmerizing, high-performance dual-band plasma waves using raw WebGL shaders (optimized through the lightweight `ogl` library). It features customizable speeds, bend intensities, rotatable views, and multi-color gradient bands.

## Usage

### Installation

<InstallBlock command="plasma-wave" />

### Basic Example

```tsx
import { PlasmaWave } from "@/registry/klarden-ui/plasma-wave";

export default function App() {
  return (
    <PlasmaWave
      colors={["#A855F7", "#06B6D4"]}
      speed1={0.05}
      speed2={0.05}
      focalLength={0.8}
      className="h-screen w-full"
    />
  );
}
```
