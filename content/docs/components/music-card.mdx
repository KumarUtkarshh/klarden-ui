The Music Card component is a premium, interactive interface that fetches live metadata from Spotify. When clicked, the album cover smoothly morphs into a rotating vinyl disc and plays an audio preview.

## Preview

<ComponentPreview name="music-card">
```tsx
import { MusicCard } from "@/components/music-card";

export default function Example() {
  return (
    <MusicCard 
      trackUrl="https://open.spotify.com/track/46yOT5iCvvQsHakl2aHhTN?si=160de9c9525a439b" 
      accentColor="#DC2626"
    />
  );
}
```
</ComponentPreview>

## Features

- **Real-time Data:** Fetches title, artist, and album art directly from Spotify.
- **Morphing Animation:** Album cover transforms into a circular vinyl record on click.
- **Audio Playback:** Plays track previews (if available) with a synchronized visualizer.
- **Dynamic Background:** Smoothly animated radial gradients based on an accent color.
- **Marquee Text:** Automatically scrolls long titles and artist names.
