The `CommandOrbit` is a sophisticated interaction hub. It features a central trigger that, when clicked, expands a set of secondary actions in a radial pattern with staggered entrance animations.

## Usage

```tsx
import { CommandOrbit } from "@/registry/klarden-ui/command-orbit";
import { MessageSquare, Share2, Mail, Settings } from "lucide-react";

const actions = [
  { icon: MessageSquare, label: "Chat", color: "text-blue-500" },
  { icon: Share2, label: "Share", color: "text-purple-500" },
  { icon: Mail, label: "Email", color: "text-rose-500" },
  { icon: Settings, label: "Config", color: "text-zinc-500" },
];

export default function Example() {
  return (
    <CommandOrbit actions={actions} radius={80} />
  );
}
```

## Features

- **Radial Layout**: Automatically calculates positions based on the number of actions provided.
- **Physics-First Motion**: Uses spring dynamics for the expansion and "wobble" effects.
- **Hover tooltips**: Integrated dark, pill-shaped tooltips for each orbiting action.
- **3D Perspective**: Includes a subtle perspective transform that follows the cursor.

## Properties

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `actions` | `OrbitAction[]` | `defaultActions` | Array of actions to display in the orbit. |
| `radius` | `number` | `80` | The distance from the center to the orbiting items. |
| `showLabels` | `boolean` | `true` | Whether to show tooltips on hover. |
| `className` | `string` | - | Additional CSS classes for the container. |
